### 问题一：

```js
// 输出啥，为什么
Promise.resolve().then(() => {
  console.log(0)
  return Promise.resolve(4)
}).then((res) => {
  console.log(res)
})

Promise.resolve().then(() => {
  console.log(1)
}).then(() => {
  console.log(2)
}).then(() => {
  console.log(3)
}).then(() => {
  console.log(5)
}).then(() =>{
  console.log(6)
})
```

题目代码输出的结果是 `0 1 2 3 4 5 6`

首先要知道如果有多个fulfilled promise实例，同时执行then链式调用，then会交替执行，防止一个promise占据太久空间
