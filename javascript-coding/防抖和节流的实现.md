# 前言：

防抖和节流在优化项目中起到重要的作用，可以避免频繁触发事件。这里的实现参考underscore的中文文档

（https://underscorejs.net/#debounce）

## 防抖：

原理：事件响应在触发一段时间才执行，在此时间内重新调用则重新计算时间，当预定的时间内没有再次调用才会执行。

#### 使用：

```js
var count = 1;
var container = document.getElementById('container');

function getUserAction(e) {
    container.innerHTML = count++;
    console.log(this)
    console.log(e)
};

container.onmousemove = getUserAction; // 未使用防抖时频繁地触发
container.onmousemove = debounce(getUserAction, 1000) // 现在随你怎么移动，反正你移动完 1000ms 内不再触发
```

#### 实现方法：

我们要实现一个`debounce`函数，利用定时器，我们可以在`debounce`函数返回一个函数，在每次调用的时候先清除定时器，然后在启动定时器，其次我们还要注意`this`的问题， 如果我们在 `getUserAction` 函数中 `console.log(this)`，在不使用 `debounce` 函数的时候，`this` 的值为： 

```js
<div id="container"></div>
```

 但是如果使用我们的 `debounce` 函数，this 就会指向 Window 对象！ 所以要在返回函数的开头保存this。

 JavaScript 在事件处理函数中会提供事件对象 event.

#### 实现代码（基础版）：

```js
const debounce = function(func, wait) {
  let timeout, result
  let debounced = function(){
    // 保存this
    let self = this
    // 获取参数
    let args = arguments
    // 清除定时器
    clearTimeout(timeout)
    timeout = setTimeout(function() {
		result = func.apply(self, args)
    }, wait)
      return result
  }
  return debounced
}
```

#### 进阶版实现：

增加了立即执行功能和事件取消功能。

```js
/**
 * 
 * @param {*} func :触发函数
 * @param {*} wait ：等待时间
 * @param {*} immdeiate：是否立即调用
 * 这里主要决解四个问题, 两个优化：
 * 1、延迟出发函数
 * 2、使this指向调用的对象
 * 3、event的指向鼠标事件
 * 4、立即调用参数
 * 5、获取result
 * 6、事件取消
 */
function debounce(func, wait, immdeiate) {
    let timeout, result
    let debounced = function(){
        // 改变执行函数内部this的指向
        let self = this
        // 获取event
        let args = arguments
        clearTimeout(timeout)
        if(immdeiate) {
            let callNow = !timeout
            timeout = setTimeout(() => {
                timeout = null
            }, wait)
            // 立即执行
            if(callNow) result = func.apply(self, args)
        }else{
            // 不会立即执行
            timeout = setTimeout(function(){
                result = func.apply(self, args)
            },wait)
        }
        return result
    }
    debounced.cancel = function(){
        clearTimeout(timeout)
        // 闭包避免内存泄漏
        timeout = null

    }
    return debounced
}
```

#### 应用场景:

- scroll事件滚动触发
- 搜索框输入查询
- 表单验证
- 按钮提交事件
- 浏览器窗口缩放

## 节流：

#### 	原理：

​				如果持续触发事件片，每隔一段时间，只执行一次事件

#### 实现方法：

这里有两种方法，使用时间戳和定时器

#### 使用时间戳：

 让我们来看第一种方法：使用时间戳，当触发事件的时候，我们取出当前的时间戳，然后减去之前的时间戳(最一开始值设为 0 )，如果大于设置的时间周期，就执行函数，然后更新时间戳为当前的时间戳，如果小于，就不执行。 

```js
// 不顾头（进去即触发），顾尾（最后一次不触发）
function throttle(func, wait) {
    let context, args
    // 之前的时间戳
    let old = 0
    return function () {
        context = this
        args = arguments
        // 获取当前时间戳
        let now = new Date().valueOf()
        if(now - old > wait){
            func.apply(this, args)
            old = now
        }
    }
}
```

#### 使用定时器:

```js
// 顾头（进去不立即即触发），不顾尾（最后触发）
function throttle2(func, wait) {
    let context, args, timeout
    return function(){
        context = this
        args = arguments
        if(!timeout) {
            timeout = setTimeout(function(){
                timeout = null
                func.apply(context, args)
            }, wait)
        }
    }
}
```

#### 双剑合璧

```javascript
function throttle3(func, wait) {
    let context, args, timeout
    let old = 0 // 时间戳
    let later = function () {
        old = new Date().valueOf()
        timeout = null
        func.apply(context, args)
    }
    return function () {
        context = this
        args = arguments
        let now = new Date().valueOf()
        if(now - old > wait) {
            if(timeout) {
                clearTimeout(timeout)
                timeout = null
            }
            func.apply(context, args)
            old = now
        }else if(!timeout){
            timeout = setTimeout(later, wait)
        }
    }
}
```

hooks版debounce
```js
function useDebounce(value, time) {
  const [value, setValue] = useState(value)
  useEffect(() => {
    const timeout = window.setTimeout(() => {
      setValue(value)
    }, time)
    return () => {
      clearTimeout(timeout)
    }
  }, [value, time])
  return value
}
```
